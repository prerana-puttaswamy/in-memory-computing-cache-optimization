# Boosting In-Memory Computing with Advanced Cache Techniques

## Overview
This project explores advanced cache optimization strategies for improving performance in hybrid memory systems. The study analyzes techniques such as OptiCache, MALRU, and near-memory processing.

## Key Contributions
- Comparative analysis of cache replacement policies (OptiCache, MALRU, WADD)
- Evaluation of hybrid memory architectures and near-memory processing
- Performance trade-off analysis across latency, energy efficiency, and throughput
- Study of FPGA-based dataflow cache architectures

## Tech & Concepts
- Computer Architecture
- Cache Optimization
- Distributed Systems Concepts
- Performance Analysis

## Results
- Reduced memory bottlenecks through optimized cache strategies
- Identified trade-offs between energy efficiency and performance
- Provided insights into scalable system design
